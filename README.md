## 简介

这个skill旨在辅助小说创作过程，特别是在梳理灵感、制定剧情大纲、发展角色和构建世界观时。当AI遇到不确定的情节或设定点时，它会通过对话框主动询问用户，确保故事发展完全符合用户的意图和创作愿景。

## 核心功能

- **反问式引导**：不自行脑补，通过提问确认用户意图
- **完整工作流**：从灵感收集到章节规划的完整流程
- **多平台兼容**：支持 Claude Code, OpenClaw, Codex

## 工作流程

### 第一步：灵感收集与初步概念
- 梳理初始灵感和想法
- 确定故事的基本类型（长篇/短篇/网络小说等）
- 明确故事的核心主题和基本冲突

### 第二步：剧情大纲制定
- 帮助构建故事的三幕结构
- 询问关键情节点和转折点
- 确保每个情节发展都有明确的目的

### 第三步：角色发展
- 发展主要角色的性格、动机和弧光
- 设定角色之间的关系网络
- 确定每个角色在故事中的作用

### 第四步：世界观构建
- 帮助构建故事发生的世界规则和设定
- 询问重要的历史背景、文化细节

### 第五步：章节规划
- 将大纲细化到章节级别
- 规划每章的核心事件和目的

## 交互原则

1. **主动确认原则**：遇到不确定点时，通过对话框询问用户意图
2. **多选项提供**：提供合理选项供用户选择
3. **建议后确认**：先提供建议，再询问用户是否接受
4. **渐进式细化**：从大框架到细节逐步深入
5. **意图尊重原则**：所有建议都服务于用户的创作意图
